# Ola Ride Data Analysis 🚖📊

This project analyzes Ola ride booking data using **SQL and Power BI** to generate insights about ride patterns, customer behavior, and booking performance.

## 📂 Project Structure

```
ola-ride-data-analysis
│
├── Dashboard
│   └── ola dashboard.pbix
│
├── DataSet
│   └── BookingsOLA.csv
│
├── Sql
│   └── ola.sql
│
└── README.md
```

## 🛠 Tools & Technologies

* SQL
* Power BI
* CSV Dataset
* Data Visualization

## 📊 Key Analysis Performed

* Successful ride bookings analysis
* Average ride distance by vehicle type
* Top 5 customers with highest bookings
* Ride cancellations by customers and drivers
* Driver rating analysis
* Average customer rating per vehicle type
* Total booking value from successful rides
* Payment method insights (UPI etc.)

## 📈 Dashboard Insights

The Power BI dashboard provides interactive visualizations including:

* Booking status distribution
* Vehicle type performance
* Ride distance analysis
* Customer rating insights
* Revenue from bookings

## 📌 Key Business Insights

* Identified the most used vehicle types
* Found top customers contributing to bookings
* Analyzed cancellation patterns
* Evaluated driver performance using ratings

## 🚀 How to Use

1. Download the dataset from the **DataSet** folder
2. Run the SQL queries from the **Sql** folder
3. Open the **Power BI dashboard** to explore visual insights
